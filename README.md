Online Food Delivery System

 An Online Food Delivery System developed to simplify the process of ordering food from restaurants through a digital platform.
This project allows users to browse menus, place orders, manage deliveries, and track order status efficiently.

 Project Overview

The system connects customers, restaurants, and delivery services in a single platform.
It enables seamless food ordering, payment handling, and order tracking using a structured software architecture.

Features

User registration and login
Browse restaurants and food items
Add items to cart
Place and manage orders
Order status tracking
Admin management of menu and orders
Exception handling and validations

Technologies Used

Java
OOP Concepts
Collections / JDBC (if database used)
MySQL / Oracle (optional backend database)
Eclipse / IntelliJ IDE
